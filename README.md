# Wildfires-and-Insurance-County-Level-Analysis-of-Voluntary-Homeowner-Nonrenewal-Rates-in-California

Overview
As wildfires have intensified across California, insurers have increasingly chosen not to renew homeowner policies in high-risk areas. This project investigates whether there is a statistically significant correlation between wildfire incidents and insurance nonrenewal rates at the county level.

Tools & Technologies
Python (Pandas, Statsmodels, Matplotlib) — data cleaning, linear regression, and visualization
QGIS — choropleth mapping by joining county-level insurance datasets with California shapefiles

Key Findings
Performed linear regression tests to evaluate correlations between wildfire frequency and nonrenewal rates across California counties
Created choropleth maps visualizing regional patterns in insurance nonrenewal rates
Findings were published in the Journal of Academic Inquiry, Fall 2025 edition
